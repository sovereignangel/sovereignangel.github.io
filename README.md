# Thesis Engine

Personal dashboard for tracking daily progress, reward scoring, and thesis coherence.

## Documentation
- [PRD](THESIS_ENGINE_PRD.md) - Complete product specifications
- [Philosophy](THESIS_ENGINE_PHILOSOPHY.md) - Design rationale and behavioral rules
- [Reward Function](REWARD_FUNCTION_ROADMAP.md) - Current implementation guide
- [Personal Context](LORI_PERSONAL_CONTEXT.md) - Personal configuration

## Quick Start
```bash
npm run dev      # Development server at localhost:3000
npm run build    # Production build
```

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Database:** Firebase Firestore
- **Auth:** Firebase Auth with Google OAuth
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **AI:** Google Gemini API
